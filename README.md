# CSC2001F Assignment 1



Here we compare the search performance of two data structures

(unsorted array and binary search tree)



## Files

- `PlaceNameEntry.java` - Stores a single place record

- `PlaceNameArray.java` - Array-based data structure

- `PlaceNameBST.java` - Binary Search Tree data structure

- `PlaceSearchArray.java` - Interactive menu for array search

- `PlaceSearchBST.java` - Interactive menu for BST search

- `PlaceExperiment.java` - Compares performance of both structures



## How to Run

Compile:

    make



Run interactive array search:

   java PlaceSearchArray



Run interactive BST search:

   java PlaceSearchBST



Run performance experiment:

   java PlaceExperiment



Clean compiled files:

   make clean



## Data Files Required

- `SAPlaceNames.csv` - Place name dataset

- `SAPlaceNamesOptimal.txt` - Optimal BST insertion order

- `SearchQueries.txt` - Queries used in the experiment
